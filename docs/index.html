Automatic cleanup

How setup works

Invite YapHub, approve the requested permissions, then run one setup command in your Discord server. No cloning, no self-hosting, no code required.

Add YapHub to your server

Use the Add to Discord button and approve the requested permissions.

Run setup in Discord

Use /yap setup and select the category where temporary voice rooms should be created.

Members join the lobby

When someone joins the configured lobby, YapHub creates their temporary room and moves them into it.

Empty rooms clean themselves up

When the last member leaves a YapHub-created room, the bot removes it.

Hosted bot, simple install

YapHub is offered as an installable Discord bot. Server owners only need to invite it and configure their voice lobby. The bot service is managed separately so your community does not need to run its own copy.

Common commands

/yap setup category:#GENERAL VOICE

/yap config

/yap profile create name:"Lower Div Yaps" category:#SMITE-LOWER-DIV lobby_name:"Lower Div Join to Yap"

Requested permissions

YapHub asks for voice and channel-management permissions so it can create temporary rooms, move members into them, and clean them up later. It should not need full Administrator access.

Built narrow on purpose

YapHub is not trying to be a mega-bot. It focuses on temporary voice channels, category-scoped join lobbies, and keeping Discord servers from turning into permanent voice-channel soup.
